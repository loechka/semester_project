Class GameObject¶

Module contains GameObject class.

class game_object.GameObject(x: int, y: int, w: int, h: int, speed: tuple = (0, 0))¶

GameObject class based on pygame.Rect.

This is a simple square object with bounds properties, which also has speed and can move.

Parameters:
  • x – left coordinate
  • y – top coordinate
  • w – object width
  • h – object height
  • speed – object speed

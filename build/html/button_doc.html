class button.Button(x: int, y: int, w: int, h: int, text: str, on_click=<function Button.<lambda>>, padding: int = 0, text_color: tuple = (255, 255, 255))¶

Button object class, based on GameObject class.

Every Button object is a rectangle that can handle mouse events.

Parameters:
  • x – left coordinate
  • y – top coordinate
  • w – object width
  • h – object height
  • text – text on the button
  • on_click – mouse click event
  • padding – text padding
  • text_color – text color

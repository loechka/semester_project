Class Bonus¶

Module contains Bonus class.

class bonus.Bonus(x: int, y: int, w: int, h: int, color: tuple, speed: int, one_type: str = 'default', good: bool = True, type: int = 0, seen: bool = True)¶

Bonus object class, based on GameObject class.

There are two types of bonuses: good - star, bad - bomb. Every good or bad bonus can have two different effects: change amount of lives, change charecter size.

Parameters:
  • x – left coordinate
  • y – top coordinate
  • w – object width
  • h – object height
  • color – object color
  • speed – object speed
  • one_type – key for generating same bonuses ‘deafult’ - use param good, ‘star’ - star, ‘bomb’ - bomb
  • good – is bonus good - True - star, False - bomb (default True)
  • type – bonus type: 0 - +/- lives, 1 - change size (default 0)
  • seen – is bonus seen (default True)

Class Wall¶

Module contains Wall class.

class wall.Wall(x: int, y: int, w: int, h: int, color: tuple, speed: int, seen: bool = True)¶

Wall object class, based on GameObject class.

Every class object is a moving left rectangle.

Parameters:
  • x – left coordinate
  • y – top coordinate
  • w – object width
  • h – object height
  • color – object color
  • speed – object speed
  • seen – is object seen (default True)
